# Rock Paper Scissors Challenge

## Basic Rules of the Game

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock

Instructions to run the app:
----------------------------
```sh
$ git clone git@github.com:[USERNAME]/rps-challenge.git
$ cd rps-challenge
$ bundle
$ rackup
```
